# 金融科技創新與應用｜期末報告

**以規模(Size)、帳面市值比(Book-to-Market)、動能(Momentum)當因子，運用機器學習預測資訊係數(IC)並建構台股投資組合**

本報告為東吳大學「金融科技創新與應用」課程（指導教授：鄭宏文）的期末報告，依照課程所教授的方法論，利用 OLS、隨機森林、類神經網路等機器學習模型，對台灣股市 26 年（1999–2025）資料進行因子預測與投資組合回測。

---

## 資料來源

- **資料庫**：TEJ Pro（臺灣經濟新報），由東吳大學授權使用
- **時間跨度**：1999 年 01 月 — 2025 年 02 月
- **使用欄位**：收盤價（元）、市值（百萬元）、股價淨值比-TEJ

---

## 因子定義

以 Fama-French 三因子理論為基礎，使用以下三個因子：

| 因子 | 定義 | 公式 |
| :--- | :--- | :--- |
| **Size（規模）** | 市值取對數，代表公司規模 | `log10(市值)` |
| **BM（帳面市值比）** | 衡量股票是否被低估或高估 | `1 / 股價淨值比` |
| **Momentum（動能）** | 過去 12 個月對數報酬 | `ln(P_{t-1} / P_{t-12})` |
| **IC（資訊係數）** | 因子值與下期報酬的相關係數，為預測目標 | `corr(X_t, R_{t+1})` |

---

## 使用模型

- **OLS**：線性迴歸，作為基準模型
- **RF**：隨機森林（100 棵決策樹）
- **NN1–NN5**：不同架構的類神經網路
- **XGBoost**：極限梯度提升
- **GLM**：廣義線性模型
- **GBRT**：梯度提升迴歸樹

---

## 結果

回測期間：2014 年 01 月 — 2025 年 01 月，研究樣本共 964 檔台灣上市櫃股票，依預測 IC 值大小排序後切分為 10 至 964 等分進行多空回測。

### 樣本外 R²

衡量模型在未參與訓練資料上的預測能力（> 0 代表有預測能力）：

| 模型 | 樣本外 R² |
| :--- | :---: |
| OLS | -3.789 |
| RF | **0.035** |
| NN1 | -2.269 |
| NN2 | **0.052** |
| NN3 | 0.023 |
| NN4 | 0.033 |
| NN5 | **0.051** |
| XGBoost | -0.159 |
| GLM | -0.643 |
| GBRT | -0.307 |

所有機器學習模型的樣本外 R² 均優於 OLS。NN2（0.052）、NN5（0.051）、RF（0.035）為唯三正值模型，代表具備實際預測能力。

### 夏普比率（大盤基準：0.174）

| 模型 | 10 等分 | 50 等分 | 100 等分 | 241 等分 | 964 等分 |
| :--- | :---: | :---: | :---: | :---: | :---: |
| OLS | 0.096 | 0.043 | 0.087 | 0.101 | -0.042 |
| RF | **0.340** | **0.422** | **0.393** | 0.248 | 0.184 |
| NN1 | 0.119 | -0.023 | -0.079 | -0.070 | -0.077 |
| NN2 | 0.381 | 0.273 | 0.179 | **0.231** | 0.086 |
| NN3 | 0.352 | 0.268 | 0.176 | 0.161 | 0.135 |
| NN4 | 0.326 | 0.198 | 0.128 | 0.090 | 0.106 |
| NN5 | 0.339 | 0.187 | 0.099 | 0.130 | 0.131 |
| XGBoost | 0.209 | 0.193 | 0.205 | 0.106 | 0.029 |
| GLM | -0.080 | -0.092 | -0.033 | -0.067 | -0.021 |
| GBRT | 0.277 | 0.359 | 0.397 | **0.248** | **0.184** |

10～107 等分的夏普比率整體表現最佳，至少有 5 個模型高於大盤。GBRT 在 20～482 等分中表現最強，平均約 0.35～0.40；RF 緊隨其後。GLM 在幾乎所有等分中為負值，表現最差。

### 勝率

| 模型 | 10 等分 | 50 等分 | 100 等分 | 241 等分 | 964 等分 |
| :--- | :---: | :---: | :---: | :---: | :---: |
| OLS | 50.7% | 45.5% | 50.7% | 55.2% | 51.5% |
| RF | 53.0% | 58.2% | **61.2%** | **62.7%** | 58.2% |
| NN1 | 46.3% | 43.3% | 43.3% | 45.5% | 48.5% |
| NN2 | 51.5% | 56.0% | 53.0% | 59.0% | 56.7% |
| NN3 | 49.3% | 53.7% | 54.5% | 58.2% | 56.0% |
| NN4 | 51.5% | 50.7% | 52.2% | 54.5% | 56.7% |
| NN5 | 50.0% | 50.7% | 53.7% | 58.2% | **59.0%** |
| XGBoost | 45.5% | 46.3% | 53.0% | 51.5% | 53.7% |
| GLM | 38.8% | 43.3% | 44.0% | 39.6% | 43.3% |
| GBRT | 48.5% | **61.2%** | 59.7% | 53.7% | 50.0% |

50～241 等分的勝率整體較佳，部分模型可達 60% 以上。RF 在各等分中勝率最為突出，GBRT 在中段等分（50～137）居第一；NN1 與 GLM 在所有等分中均未能穩定勝過大盤。

### 累積報酬

- **最佳表現**：964 等分下 RF 累積報酬突破 6，為所有模型中最高
- **整體趨勢**：RF 與 GBRT 在大多數等分中持續領先，累積報酬多落在 3～6 之間
- **最差表現**：10 等分的所有預測模型累積報酬均低於 2；NN1 與 GLM 在各等分中長期落後大盤

## 操作流程

```
資料下載（TEJ Pro）
    ↓
資料處理（Excel）：計算 Size / BM / Momentum 因子值與月報酬
    ↓
計算真實 IC 值
    ↓
機器學習模型訓練（Python）：OLS / RF / NN1–NN5 / XGBoost / GLM / GBRT
    ↓
預測 IC 值 → 依 IC 排序分組（10 / 20 / 50 等分）
    ↓
績效分析：勝率、夏普比率、累積報酬
```
